## 程序中支持三种导入身份证图片模式：
### 文件访问
通过选择图片文件进行导入
### 剪贴板导入
将剪贴板中复制的图片进行导入
### 图片拖入
通过鼠标将想要导入的图片拖入到窗体组件区域，程序会自动上传图片进行分析，个人比较推荐该方式
